<h1>Descrição</h1>

Este é um programa simples em Python que realiza uma requisição HTTP para a API da 42. Ele obtém um JSON baseado nos alunos que estão logados no laboratório da 42 Rio. O programa vai printar na tela o login e a máquina onde esta logado o estudante e no final irá dizer o total de estudantes que estão no campus da 42 Rio.

O usuário também pode escolher o formato de exibição: 'formated' (dados formatados) ou 'json' (printa um json identado no terminal).

<h1>Como usar</h1>
Execute o programa passando <strong>um argumento</strong>:
 <ul>
  <li><strong>Formato que você deseja visualizar a saída no terminal.</strong></li>
</ul> 

<h1>Exemplo</h1>
<p><code>python nome_do_arquivo.py <formated|json></code></p>
<p><code>python exercicio2.py formated</code></p>
<p>Isso exibirá o login e a máquina de todos os estudantes ativos e o total de estudantes presentes no campus.</p>
